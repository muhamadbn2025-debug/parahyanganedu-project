# Parahyangan Edu Project

## Deskripsi Project

Parahyangan Edu Project merupakan website sederhana berbasis HTML dan CSS yang digunakan untuk menampilkan informasi layanan edukasi secara terstruktur.

Project ini dibuat sebagai portfolio untuk menunjukkan kemampuan dasar dalam pengolahan data menggunakan SQL serta menampilkan data tersebut ke dalam tampilan website.

---

## Latar Belakang

Di wilayah Bandung dan sekitarnya, masih banyak masyarakat yang mengalami kesulitan dalam mengakses informasi terkait layanan edukasi seperti kursus, pelatihan kerja, dan program pembelajaran.

Selain itu, data layanan edukasi seringkali tidak terorganisir dengan baik, sehingga sulit untuk diakses oleh masyarakat yang membutuhkan.

Berdasarkan permasalahan tersebut, dibuatlah website sederhana yang dapat membantu menampilkan data layanan edukasi secara lebih terstruktur.

---

## Tujuan

* Menampilkan data layanan edukasi dalam bentuk website
* Mengolah data menggunakan query SQL
* Memahami hubungan antara database dan tampilan web
* Membangun dasar pengembangan aplikasi berbasis web


## API Integration

Dalam project ini, API belum diimplementasikan secara penuh. Namun, konsep API dipahami sebagai penghubung antara database dan frontend.

Data yang digunakan berasal dari hasil query SQL dan ditampilkan secara statis sebagai simulasi alur API.
